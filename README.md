# Cyber Quiz 🛡️

A cybersecurity quiz app with 50+ real questions across 5 categories.

## File Structure

```
Cyber-Quiz/
├── index.html      ← HTML structure (3 screens: Start, Quiz, Results)
├── style.css       ← All styling (dark hacker theme)
├── questions.js    ← Question bank (50+ real cybersecurity questions)
├── script.js       ← Quiz logic (timer, scoring, shuffle, navigation)
└── README.md       ← This file
```

## How to Run

1. Open the folder in VS Code
2. Install the **Live Server** extension (by Ritwick Dey)
3. Right-click `index.html` → **Open with Live Server**
4. Opens at `http://127.0.0.1:5500`

## Features

- 5 categories: Network Security, Cryptography, Threats & Attacks, Ethical Hacking, General Security
- Difficulty filter: All / Easy / Medium / Hard
- 15-second countdown timer per question
- Instant answer feedback with correct answer highlight
- Score, correct, wrong, skipped breakdown on results screen
